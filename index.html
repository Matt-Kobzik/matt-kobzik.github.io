Job Scraper
Built a custom Python job scraper during my active job search instead of manually checking boards every day.

The Python script fetches job listings using Greenhouse, Lever, and Ashby ATS APIs across 80+ target companies; it filters by role type and location, and scores each match 1–5 using the Anthropic API — writing results to a structured Notion database daily.

The scraper is my first foray into AI-assisted engineering, leveraging my domain expertise in recruiting and job search with my technical skills to build a tool that saves me hours of manual work each week and ensures I never miss an opportunity at a target company.
